# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-21

### Added
- Initial release
- 8 content generation templates: blog, social, email, tweet-thread, ad, seo, product, youtube
- 3 LLM providers: OpenAI, Gemini, Ollama (local)
- Real-time streaming with Rich markdown rendering
- TOML configuration with environment variable overrides
- File output and clipboard support
- Template browsing and provider status commands
